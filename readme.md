# 🌟 My Little Star — Smart Learning Coach

A fun, interactive learning app for kids aged 4–5, built as a single HTML file with no dependencies.

![HTML5](https://img.shields.io/badge/HTML5-Single%20File-orange) ![JavaScript](https://img.shields.io/badge/JavaScript-Vanilla-yellow) ![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Features

### 📚 Learning System
- **60-day curriculum** across 4 subjects: ABC, Maths, Art, Science
- **8-week progressive structure** — content unlocks week by week
- **Adaptive difficulty** — auto-adjusts across 3 levels based on performance
- **Activity picker** — browse and choose activities, or tap 🎲 Random

### 🎮 8 Interactive Games
- 🔢 Number Match | 🔤 Build a Word | 🍎 How Many? Count it!
- 🎨 Colour Mixing Lab | 🔺 Tap the Right Shape
- 🧠 Memory Flip Cards | 🔤 Find the Letter | 🐶 Animal Sounds Match

### 🏃 Physical Activities
- Animated stick-figure exercises every 2 days
- Jumping Jacks, Squats, Stretches, Dance Party, Bear Crawl & more
- "I did it!" real-world confirmation button

### 🦉 Smart Mascot
- Context-aware owl companion with speech bubbles
- Tracks streaks: "3 in a row! 🔥"
- Adapts to wrong answers: "Let's try something easier!"
- Celebrates milestones and greets by time of day

### 🔊 Voice System
- Full text-to-speech: reads title, description, learning points, and all steps
- Voice picker in Parent Dashboard — test and select from available voices
- Toggle sound on/off

### 🎨 Creative Zone
- Full drawing canvas with touch support
- 9-color palette, 4 brush sizes, stickers, eraser
- 💾 Save artwork as PNG

### 🎁 Visual Rewards
- Star Shop with hats, characters, and themes
- Hats appear on mascot, characters swap the owl, themes change app colors
- Tap to equip/unequip

### 👨‍👩‍👧 Parent Dashboard
- Activities done, stars earned, streak tracking
- Smart insights: "💪 Strongest: ABC" / "🌱 Needs practice: Maths"
- Voice selection, bedtime mode, difficulty display

### 🌙 Bedtime Mode
- Dark purple theme for evening use
- Calming sound on activation

### Other Features
- ⏭️ Skip any activity (comes back tomorrow)
- ⏱️ 5-minute focus timer
- 🎯 Weekly challenges
- 📋 Daily story mode
- 📸 "Show me — Done!" real-world capture
- 🔄 Pause/Resume — picks up where you left off

## 🚀 How to Use

1. Open `index.html` in any modern browser
2. That's it! No server, no build step, no dependencies

Works on phones, tablets, and desktops. All data saves to localStorage.

## 📱 Compatibility

- ✅ Chrome, Safari, Firefox, Edge
- ✅ iOS Safari, Android Chrome
- ✅ Works offline after first load
- ✅ Responsive: 320px to 1200px+
- ✅ Safe area support for notched phones

## 🛠️ Tech Stack

- **Single HTML file** — everything in one file
- **Vanilla JS** — no frameworks, no build tools
- **CSS3** — animations, gradients, responsive design
- **Web Speech API** — text-to-speech
- **Web Audio API** — sound effects
- **localStorage** — persistent progress tracking
- **SVG animations** — stick-figure exercise demos
